Frequently Asked Questions

We have worked hard to make this an easy to use interface, but just in case...

How does this work?

The photogrammetry rig will take a succession of pictures at various angles of a model in order to provide a comprehensive data set from which a 3D model can be constructed.

What software is best to create my model with?

That's a tough one! There are lots of free and inexpensive software packages out there. We like Agisoft, there's a student discount for the Personal edition. Check it out.

How did you build this?

The Photogrammetry rig is based on the 3D model created by Tho Mas. You can find the files on Thingyverse here. Important notes if you decided to 3D print the files:

  • The rail adapters (#1a & #1b) need to be scaled 10x
  • The camera holder (#7) is printed twice and must be "mirrored"
The s/w system that runs the rig was custom developed using Python on the Raspberry Pi.
